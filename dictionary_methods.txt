Python. Методи для словників (Dictionary methods)
Python має набір вбудован

clear() — Видаляє всі елементи зі словника

copy() — Повертає копію словника

fromkeys() — Повертає словник із зазначеними ключами та значенням

get() — Повертає значення вказаного ключа

items() — Повертає список, що містить кортеж для кожної пари значень ключів

keys() — Повертає список з переліком ключів словника

pop() — Видаляє елемент із зазначеним ключем

popitem() — Видаляє останню вставлену пару ключ-значення

setdefault() — Повертає значення заданого ключа. Якщо ключа не існує: вставляє ключ із вказаним значенням

update() — Оновлює словник заданими парами ключ-значення

values() — Повертає список всіх значень у словнику
