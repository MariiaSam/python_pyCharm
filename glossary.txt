assignment statement (операція присвоювання) 

assignment token (оператор присвоювання) — знак рівності (=) 

block (блок) — частина послідовних виразів з однаковим відступом

body (тіло) — вирази у циклі, вирази з однаковим рівнем відступу є частиною одного блоку / комплексу і складають тіло блоку

boolean expression (булевий вираз) — логічна операція, результатом якої може бути тільки True або False. Булевий вираз — це результат булевої функції

boolean function (булева функція) — фрагмент коду, який запитує деякі дані. Її результатом може бути тільки True або False

comment (коментар)

comparison operator (оператор порівняння) — один з операторів, який порівнює два значення.

condition (стан) — це тригер для виконання коду, який залежить від того, чи є булевий вираз істиним або хибним

conditional statement (умовна операція) — оператор, який регулює хід виконання програми в залежності від стану

control flow (керівна логіка) — різні типи операцій, такі як умовні (if, elif і else), цикли (while) і таке інше, які скеровують Python ігнорувати послідовність коду зверху вниз і реагувати на істинні або хибні відповіді від вхідних даних, запитів на дані і таке інше

counter (контейнер counter) — змінна, яка використовується щоб підрахувати щось, що зазвичай починається з нуля і збільшується покроково у тілі циклу

data structure (структура данних) — це спосіб зберігання та організації данних для полегшення їх використання. У Python дані можуть бути структуровані у вигляді списків, словників, кортежів або множин

data type (тип данних) — це спосіб класифікації конкретних типів данних і як вони можуть бути використані у виразах і операціях. Приклади типів данних: текстові рядки, цілі числа і числа з плаваючою комою

dictionary (словник) — це колекція пар ключ-значення, яка зіставляє разом типи даних

element (елемент) — одне зі значень списку (або іншої послідовності)

escape (екранування) — символ зворотнього слешу \\ використовується у Python для позначення спеціальних символів

expression (вираз) — це спосіб представлення значення, яке може бути збережене для використання пізніше. Це комбінація значень, змінних, операторів і викликів функцій. Вираз відрізняється від операції тим, що він є просто значенням, а операція є інструкцією для Python щось зробити.


float (число з плаваючою комою) — тип даних Python, в якому зберігається число з плаваючою комою. Коли його надрукувати в стандартному форматі, число буде виглядати як десяткове число. У Python число з плаваючою комою округлюється в меншу сторону, тому 1.25 буде округлене до 1.2, а не до 1.3, як можна було вважати.

for loop (цикл for) — операція у Python, яку зручно використовувати при роботі зі списками елементів. Цикл повторює операцію знову і знову, поки не пройде по всім можливим елементам

index (індекс) — цілочислова змінна або значення, яке показує кожен елемент (включаючи пробіли) текстового рядка або списку. Індексація дозволяє оперувати даними більш легко, наприклад, знайти передостаннє значення у довгому списку. Пам’ятайте, що індексація завжди починається з 0

infinite loop (нескінченний цикл) — цикл, в якому ніколи не досягається умова його припинення.
int (число) — тип даних Python, в якому зберігається позитивне або негативне ціле число.


input prompt string (рядок вводу) — надає користувачеві підказки щодо типу значення, яке слід ввести для змінної з назвою name.
name = input(“What’s your name? ”)

key (ключ) — елемент данних, об’єднаний зі значенням у словнику. Ключ служить для пошуку значення у словнику.
key-value pair (пара ключ-значення) — пара значень у словнику. Пошук значень у словнику виконується саме за ключем.
keyword (ключове слово) — зарезервоване слово, яке використовується програмою для парсингу (розуміння) коду; ви не можете використовувати ключові слова, такі як if, def, input і while у якості назв змінних або назв ключів.
len (довжина) — скорочення від слова «length», вказує Python порахувати якусь дистанцію. Це може бути кількість літер у слові або кількість елементів в об’єкті.
# скільки літер у слові "hello"?
var1 = len(“hello”)
var1 = 5
# скільки пунктів у цьому списку слів привітання?
var2 = [“hi”, “hello”, “ola”]
var2 = 3

list (список) — структура даних, яка є колекцією об’єктів (елементів).
menu_items = [“wrap”, “salad”, “sandwich”, “soup”]

logical operator (логічний оператор) — один з операторів, який поєднує булеві вирази: and, or та not.
loop (цикл) — операція або група операцій, які виконуються повторно, доки не буде досягнуто умови припинення циклу

nested list (вкладений список) — список, який є елементом іншого списку

nested loop (вкладений цикл) — цикл у тілі іншого циклу

operator (оператор) — спеціальний символ, який представляє прості обчислення, такі як складання (+), множення (*), віднімання (-), ділення (/) або складання текстових рядків (+)

parameter (параметр) — спеціальний тип змінної, який використовується для передачі даних у функцію. В одній функції може бути декілька параметрів

printfunction (виведення на екран) — функція в коді Python, що використовується для виведення на екран команди

range (діапазон) — вбудована у Python функція, яка генерує послідовність цілих чисел. Це особливо зручно, коли нам треба написати цикл, який має виконуватись конкретну кількість разів.
reassignment (перепризначення) — призначення іншого значення тій самій змінній в процесі виконання програми

return value (повернення значення) — значення, яке повертається в результаті виклику функції.

slice (зріз) — частина текстового рядка (підрядок)

statement (операція) — інструкція, яку інтерпритатор Python може виконати

tuple (кортеж) — тип даних, який містить послідовність елементів різних типів, таких як список. Вміст кортежу не може бути змінений

type conversion function (функція перетворення типів) — функція, яка може перетворювати значення даних одного типу в інший.

value (значення) — рядок (набір симфолів) або ціле число, яке може бути збережене у змінній.

variable (змінна) — назва, яку Python зберігає в пам’яті для пошуку і використання в подальшому. Змінна завжди має почитатись з літери і писатись в нижньому регістрі. Декілька слів у назві змінної мають розділятись нижнім підкресленням. За найкращими практиками промування назва змінної має описувати її призначення в програмі, роблячи програму самозадокументованою. Також дивіться: зарезервовані слова.

while loop (цикл while) (іноді називають невизначеною ітерацією) — цикл, який продовжує виконуватись доки не буде виконана деяка умова TRUE або FALSE. Цикл while і булеві значення використовуються разом.

whitespace (символ пробілу) — будь-який символ, який зміщує курсор без друку видимих символів.
