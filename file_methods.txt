Python. Методи для файлів (File methods)
Python має набір методів, доступних для роботи з файловими об'єктами.

close() — Закриває файл

detach() — Повертає відокремлений необроблений потік з буфера

fileno() — Повертає число, яке представляє потік з точки зору операційної системи

read() — Повертає вміст файлу

readable() — Повернення даних, чи можна прочитати файловий потік чи ні

readline() — Повертає один рядок з файлу

readlines() — Повертає список рядків з файлу

seek() — Зміна позиції файлу

seekable() — Повернення даних, чи дозволяє файл змінювати його позицію

tell() — Повертає поточне розташування файлу

truncate() — Змінює розмір файлу до вказаного розміру

writable() — Повертає дані, чи можна виконати запис до файлу, чи ні

write() — Записує заданий рядок у файл

writelines() — Записує список рядків у файл
