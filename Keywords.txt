Python. Ключові слова (Keywords)
Python має набір ключових слів, які являють собою зарезервовані слова, що не можуть бути використані як імена змінних, імена функцій або будь-які інші ідентифікатори.


and — Логічний оператор

as — Для створення псевдоніму (аліасу)

assert — Для усунення неполадок

break — Для розриву циклу

class — Д ля визначення класу

continue — Для переходу до наступної ітерації циклу

def — Для визначення функції

del — Для видалення об'єкту

elif — Використовується в умовних виразах, так само як і else if

else — Використовується в умовних виразах

except — Використовується з винятками. Пояснює, що робити, коли виникає виняток

False — Булеве значення, результат операцій порівняння

finally — Використовується з винятками. Позначає блок коду, який буде виконано незалежно від того, чи присутній виняток, чи ні.

for — Для створення циклу for

from — Для імпорту окремих частин модуля

global — Для оголошення глобальної змінної

if — Для створення умовного виразу

import — Для імпорту модуля

in — Перевірка наявності значення у списках, кортежах тощо.

is — Для перевірки рівності двох змінних

lambda — Для створення анонімної функції

None — Являє собою нульове значення

nonlocal — Для оголошення нелокальної змінної

not — Логічний оператор

pass — Нульовий вираз — твердження, яке нічого не зробить

raise — Для створення винятку

return — Для завершення функції та повернення значення

True — Булеве значення, результат операцій порівняння

try — Для виконання спроби, окрім виразів

while — Для створення циклу while

with — Використовується для спрощення обробки винятків

yield — Використовується для повернення списку значень з функції.
